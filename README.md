# Shussekikunとは

Shussekikunはジムや習い事教室の店舗オーナーが、会員の出席を管理できるWebシステムです。

## 構築手順

- 要件定義
- 構造設計
- デザイン
- インフラ構築
- DB設計
- 開発
- テスト
- 公開
